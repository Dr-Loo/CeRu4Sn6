# SFIT Topological Dome Validation — CeRu₄Sn₆

This repository contains experimental data and analysis scripts designed to test the **SFIT theoretical prediction** of a *topological dome* in the anomalous Hall conductivity of **CeRu₄Sn₆** under varying pressure and magnetic field conditions.

Key results confirm:
- The existence of a topological dome at **(P, B) = (0, 0)**
- Suppression of the topological response at higher pressures

These findings are consistent with predictions from the **SFIT framework**.
